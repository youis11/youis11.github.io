FIRE & ICE

PLATFORMER made with SDL

Fire & Ice is a jump ‘n run platformer game.You will be able to play with a little unicorn being able to jump, run, change between forms, kill enemies and even die falling into the venom! Make sure to save the game in crucial situations. Fire goes with fire and ice goes with ice. Don’t forget that! This project consists of develop a platform game and program it from the beggining using SDL, TILED and XML. So the whole group is going to program the game but we had to take some responsabilities that were given to each of the group.

WORK RESUME
  • 1st assignment
    • Create Levels using Tiled
    • Search Art such us tiles, characters and music/sfx
    • Volume functions
  • 2nd assignment
    • Entity + Entity Manager (previously defining an UML)
    • Enemy class
    • Add Brofiler
  • 3rd assignment
    • GUI structure and elements (previously defining an UML)
    • Create mainmenu level
    • Create GUI
    • Coin entity
    • Website
