DUNGEON TOUR

PRODUCER

Dungeon Tour is a project currently in development by Tequila Works, defined as a midcore cooperative party game for up to four players. Players must manage a horde of tourists in chaotically generated dungeon-like environments, cooperating with the rest of the team.
The visual style, narrative, and gameplay are built around dark and absurd humour, in a conceptual mix between Overcooked and Dungeon Keeper.
Genre: Cooperative multiplayer - party game
Development status:
Project in development, with vertical slice available
Engine: Unreal Engine
